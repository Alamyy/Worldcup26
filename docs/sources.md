# Sources

The CSV files in this repository are based on publicly available FIFA Training Centre post-match reports for FIFA World Cup 2026.

Each match row includes a `source_url` field that points to the corresponding source report.

## What This Repository Publishes

- CSV data tables
- Table metadata
- Data dictionary
- Join/schema reference

## What This Repository Does Not Publish

- Original report PDFs
- Player images
- Scraped media
- Dashboard assets

## Attribution

When using this data, please attribute:

- FIFA Training Centre as the source of the original match reports
- This repository as the source of the published open CSV dataset

This project is independently maintained and is not affiliated with FIFA.
